# LogiFlow

## Як запустити

```bash
cd "Your path"
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn backend.main:app --reload
```

Після запуску відкрий у браузері:

```text
http://127.0.0.1:8000
```

## Яку проблему ми вирішуємо

У логістиці складно швидко зрозуміти:
- які точки потребують допомоги першими
- які машини куди мають їхати
- скільки буде коштувати маршрут
- як не витрачати зайві кілометри та гроші

## Що робить наша програма

LogiFlow допомагає побачити всю мережу на одному екрані та швидко прийняти рішення:
- генерує логістичну мережу з заводами, складами й магазинами
- будує маршрути для машин
- показує критичні точки, які треба закривати першими
- візуалізує карту мережі й маршрути
- пояснює, як формується вартість перевезень
